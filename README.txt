== Description ==

Wordpress News Ticker  Plugin Version: 1.0 This is first version of this plugin having options for
1]Width of the News Ticker.
2]Border Size and Color
3]Select Post Category to show posts in News Ticker.
4]Show Images or not in News Ticker.

== Installation ==

1. Download the zipped plugin file to your local machine.
2. Unzip the file.
3. Upload the "wordpress-news-ticker" folder to the "/wp-content/plugins/" directory. (DO NOT CHANGE ANY FILE OR FOLDER NAMES)
4. Activate the plugin through the "Plugins" menu in WordPress.
5. Add the folowing code anywhere within your theme where you want the content gallery to be displayed: `<?php include (ABSPATH . ‘/wp-content/plugins/wordpress-news-ticker/news_ticker.php’); ?>`

== Usage ==

After you download and activate the new Wordpress News Ticker plugin, you will first want to head over to the options page which will allow you to fully customize your News Ticker width, height, and slideshow border colors and size. You will also need to enter the category name from which all the images will come also the number of posts.

There is 'cache' folder inside the wordpress-news-ticker plugin make sure you have 777 Write permission.
